# feedingfilipinos

Charity site scaffold: React frontend, Next.js backend.

Quick start (PowerShell):

1. Frontend
   cd D:\Projects\feedingfilipinos\frontend
   npm install
   npm run dev

2. Backend
   cd D:\Projects\feedingfilipinos\backend
   npm install
   npm run dev

Notes:
- Frontend uses Vite + React. It defaults to port 5173.
- Backend is Next.js and runs on port 3001 (dev script).
- Adjust Node version if needed (Next.js prefers Node 18+).
